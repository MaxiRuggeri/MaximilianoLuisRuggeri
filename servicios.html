Trabajo Practico

Internet

El Internet es una infraestructura global de redes interconectadas. Es un sistema de redes de computadoras que se comunican entre sí utilizando protocolos estandarizados, específicamente el Protocolo de Internet (IP). Esta red global permite la transmisión de datos y facilita la comunicación entre dispositivos a nivel mundial. El Internet abarca todo tipo de comunicación de datos, incluyendo correo electrónico, transferencias de archivos (FTP), servicios de chat, y muchos otros servicios y aplicaciones.

Web

La Web (o World Wide Web) es un servicio específico que se ejecuta sobre la infraestructura del Internet. Fue inventada por Tim Berners-Lee en 1989. La Web utiliza el Protocolo de Transferencia de Hipertexto (HTTP) para acceder a documentos y recursos vinculados entre sí mediante hipervínculos. Estos documentos suelen estar en formato HTML y pueden incluir texto, imágenes, videos, y otros tipos de contenido multimedia. Los navegadores web (como Chrome, Firefox, Safari) son las herramientas que usamos para acceder y navegar por la Web.
